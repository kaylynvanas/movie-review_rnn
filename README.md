# Movie Review Recurrent Neural Network
**Data Source:** http://ai.stanford.edu/%7Eamaas/data/sentiment/
## Description
This recurrent neural network in Keras uses sentiment analysis to classify a movie review as either negative or positive.

## Run
To run:
- Download the [Jupyter Notebook](https://github.com/kaylynvanas/movie-review_rnn/blob/master/Capstone%20Project%20III.ipynb) to a folder;
- Open your terminal or command prompt and change directory to the folder using `cd`;
- Enter `jupyter notebook`;
- Once Jupyter Notebook is open, click Cell -> Run All in the menu.
